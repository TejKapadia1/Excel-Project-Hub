Project Description
This project analyzes the risk and return characteristics of a diversified portfolio consisting of ten large-cap U.S. companies: Apple (AAPL), Coca-Cola (KO), Netflix (NFLX), Berkshire Hathaway (BRK.B), Walt Disney (DIS), IBM, Verizon (VZ), Walmart (WMT), General Electric (GE), and Tesla (TSLA). The objective of the project is to evaluate individual stock performance, construct diversified portfolios, and apply quantitative financial models to understand portfolio risk and return dynamics.
The analysis begins with the calculation of daily returns, annualized returns, volatility, and beta for each stock relative to the S&P 500 market benchmark. Multiple portfolio strategies are then constructed, including an equal-weighted portfolio, an optimized portfolio using Solver, and a minimum variance portfolio. In addition, several two-asset portfolios are analyzed to demonstrate diversification effects and the efficient frontier.
Portfolio performance is evaluated using key financial metrics such as expected return, standard deviation, Sharpe Ratio, portfolio beta, and Jensen’s Alpha. To assess downside risk, Value at Risk (VaR) is calculated using the variance–covariance method. The project highlights how diversification, asset allocation, and optimization techniques can improve portfolio efficiency and risk-adjusted performance.
